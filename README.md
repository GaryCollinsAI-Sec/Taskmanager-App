# Task Manager App 📝

A streamlined, cross-platform mobile task management application built with **React Native** and **TypeScript**. Designed for efficiency, performance, and clean code architecture.

## 🚀 Features
* **Task Management**: Create, view, and delete tasks in real-time.
* **TypeScript Integration**: Strong typing for data structures to ensure code reliability and minimize runtime errors.
* **Responsive UI**: Built using React Native `StyleSheet` for a consistent experience across iOS and Android.
* **Optimized Rendering**: Leverages `FlatList` for efficient list management and performance.
* **State Management**: Uses React `useState` hooks for clean, reactive component updates.

## 🛠️ Tech Stack
* **Framework**: React Native (via Expo)
* **Language**: TypeScript
* **State Management**: React Hooks
* **UI/Styling**: React Native StyleSheet
* **Platform**: iOS / Android
